# v1.0.0 — first public release

EasyFold is a web UI that makes [AlphaFold 3](https://github.com/google-deepmind/alphafold3) and [Boltz-2](https://github.com/jwohlwend/boltz) protein structure prediction usable by experimental biologists who don't code. You paste a sequence (or look it up by UniProt accession / PDB ID), add ligands and modifications, click **Predict**, and 5–15 minutes later you get a 3D structure, per-residue pLDDT, PAE, and — uniquely — a natural-language explanation of what those numbers mean for *your* scientific question.

## The 3 things that make EasyFold different

- **Question-driven input.** You think in proteins, ligands, modifications, and copies — not in AF3's JSON schema. The assembly builder UI maps your scientific question to the underlying spec.
- **LLM interpretation layer.** Ask "Is this DNA-binding pocket trustworthy enough for docking?" and Claude answers in plain English, grounded in the actual pLDDT / PAE / ipTM numbers your prediction produced, with concrete next-step suggestions.
- **Two models behind one UI.** Pick AlphaFold 3 (CC-BY-NC-SA, highest reference quality) or Boltz-2 (MIT, no Google approval needed) per job. Same input shape, same result viewer.

## Try it now

**[→ huggingface.co/spaces/maiko811/easyfold-demo](https://huggingface.co/spaces/maiko811/easyfold-demo)** — no install, no GPU, no API key needed.

Three pre-computed example structures: **1TUP** (p53 DNA-binding domain), **1CRN** (crambin), and **6LU7** (SARS-CoV-2 main protease). Each comes with the Mol\* 3D viewer, pLDDT / PAE confidence charts, and the Claude interpretation panel ready for your own [Anthropic API key](https://console.anthropic.com/).

## Run real predictions in ~10 minutes

The [README Quickstart](https://github.com/maikoo811/easyfold#quickstart-run-real-predictions-in-10-minutes) takes you from clone to your first real Boltz prediction. AlphaFold 3 setup is in [`modal/README.md`](https://github.com/maikoo811/easyfold/blob/main/modal/README.md) (2–3 day Google weight-approval wait).

**Cost reality** (Modal pays per-second of H100 time):

| Path | First run | Steady-state | Storage |
|---|---|---|---|
| Boltz-2 | ~10 min (~$0.50, includes weight download) | **~30 s–5 min per job (~$0.05–0.40)** | ~$0.10/mo cache |
| AlphaFold 3 | ~10 min (~$0.50–1) | ~5–10 min (~$0.50–1) | ~$0.30/mo weights |

**No subscription, no central service, no per-API-call fee.** You pay Modal for GPU time, and Anthropic for the optional Interpret panel (~$0.01 per question, your own key).

## Real-prediction validation

The README screenshots aren't synthetic — they're from an actual Boltz-2 run against [UniProt P04637 (p53)](https://www.uniprot.org/uniprotkb/P04637/entry) with 2 copies + a `PHOSPHO@15` modification + an `HEM` ligand. Claude's interpretation correctly names the intrinsically disordered N/C termini and points to the DNA-binding domain as the high-confidence region.

## Zero-hosting OSS

- **BYOC**: inference runs in **your** Modal account. We don't have credentials, don't see your sequences, don't pay your bill.
- **BYOK**: your Anthropic API key stays in browser memory only — never sent to the EasyFold backend, never written to `localStorage`.
- **Stateless backend**: no database, no auth, no session store. Job IDs are Modal's `FunctionCall.object_id` — see [ADR 0004](https://github.com/maikoo811/easyfold/blob/main/docs/decisions/0004-jobs-api-modal-funcall-as-id.md).
- **No maintainer dependency**: if this project goes quiet, your fork keeps working — there's no central service to depend on.

## Security

Internal audit + external review (ChatGPT-based) both returned **0 critical / 0 high findings**. Input size caps at the Pydantic boundary, 5 MB request-body middleware, browser security headers (`X-Frame-Options`, `Referrer-Policy`, etc.), and bearer-secret job URLs (~131 bits of entropy). Full threat model + reporting channel in [`SECURITY.md`](https://github.com/maikoo811/easyfold/blob/main/SECURITY.md). The README's ["What leaves your machine"](https://github.com/maikoo811/easyfold#what-leaves-your-machine) table is the authoritative data-flow disclosure.

## Known constraints

- AlphaFold 3 weights need Google approval (2–3 business days). The repo can't ship them — request via [Google's AlphaFold 3 form](https://github.com/google-deepmind/alphafold3#obtaining-model-parameters).
- AlphaFold 3 is **CC-BY-NC-SA** (non-commercial). Boltz-2 is **MIT** (commercial OK). The README's "Pick a model" table maps your use case to the right choice.
- Boltz-2 silently drops post-translational modifications at this MVP. The UI surfaces this as a disabled Predict-with-Boltz button + inline tooltip when modifications are present.
